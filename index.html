About rTuner

rTuner is a basic chromatic tuner designed for Firefox OS devices, but it can run on every modern browser. The program uses the MPM algorithm for fast and robust pitch detection.

Features

  • Fast pitch detection with 3 cents accuracy.
  • Wide frequency range from A2 to A7.
  • Adjustable A4 base frequency from 400 to 480Hz.
  • Variable microphone gain settings.

Planned features

  • Pitch generator.
  • Different algorithms.
  • Wide range of adjustable settings.
